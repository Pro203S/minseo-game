# 게임 데이터

- 기존 선별 게임과 라이브 수집분은 Steam 한국 스토어 API의 한국어 설명 및 원화 정가를 사용합니다.
- 대량 수집분은 CC0로 공개된 `Steam Games Dataset 2021-2025 (65k+)`에서 인기순으로 가져옵니다.
- 대량 수집분의 가격은 데이터셋의 USD 가격에 2026-07-10 기준 환율(약 1 USD = 1,505 KRW)을 적용하고 100원 단위로 반올림한 참고 가격입니다. Steam 한국 스토어의 실제 지역 가격과 다를 수 있습니다.
- 이미지는 Steam CDN의 App ID 기반 헤더 이미지 주소를 사용합니다.
- `npm run seed:games`는 중복을 건너뛰며 약 10,000개까지 채웁니다.
- `npm run seed:games:live`는 Steam 한국 스토어 API로 데이터를 보강하지만 호출 제한 때문에 장시간 실행될 수 있습니다.

게임 가격과 판매 상태는 바뀔 수 있으므로 구매 전 Steam 상점에서 다시 확인해야 합니다.
